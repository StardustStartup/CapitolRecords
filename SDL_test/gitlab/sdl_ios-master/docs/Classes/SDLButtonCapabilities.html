Overview

Provides information about the capabilities of a SDL HMI button.

@since SDL 1.0

name

  • The name of the SDL HMI button.

Required

Objective-C

@property (readwrite, strong, nonatomic) SDLButtonName _Nonnull name;

Swift

var name: SDLButtonName { get set }
